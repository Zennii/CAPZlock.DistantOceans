# Distant Oceans mod
The Distant Oceans mod adds a new Dimension to Hytale, including 11 new biomes to explore and discover.
[You can grab the mod on CurseForge here.](https://www.curseforge.com/hytale/mods/distant-oceans)

# Dependencies
DOStats is a required dependency.

https://github.com/Zennii/CAPZlock.DOStats
